# PRD Generation Prompt
Create a concise PRD that frames the problem, scope, priorities, and success criteria.

## Focus
- Lead with the problem and the value to the user.
- Keep goals, metrics, requirements, constraints, and risks tight.
- Use non-goals to keep scope honest.

## Completion Criteria
- The document is decision-oriented.
- Requirements are prioritized.
- Metrics, assumptions, and risks are explicit.
