# Launch Checklist Generation Prompt

Create a concise launch readiness checklist for a coordinated product launch.

## Storage Location

`docs/helix/05-deploy/launch-checklist.md`

## Key Requirements

- Keep it cross-functional but short.
- Include only the readiness checks needed to launch this product or service.
- Make the go/no-go criteria measurable and decision-ready.
- Avoid broad, generic launch boilerplate.

## Completion Criteria

- [ ] Product, marketing, sales, support, and compliance readiness are covered
- [ ] Each section is specific enough to execute
- [ ] Go/no-go criteria and blockers are clear
