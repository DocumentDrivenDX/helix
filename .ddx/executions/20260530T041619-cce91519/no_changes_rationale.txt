1. Operator access to a Databricks workspace with Genie Code enabled
2. HELIX skill installed at /Workspace/.assistant/skills/helix/
3. DBAUTH cookie from authenticated browser session

Procedure for operator:

  # Step 1: Verify HELIX installed
  python tests/install/genie/install.py --shared
  python tests/install/genie/verify.py

  # Step 2: Obtain DBAUTH cookie (from browser DevTools)
  # Application → Cookies → DBAUTH (copy value to /tmp/dbauth.txt)

  # Step 3: Run the test harness with recording
  export DATABRICKS_HOST=https://adb-XXXX.azuredatabricks.net
  export DATABRICKS_WORKSPACE_URL=/genie/notebooks/...
  export DBAUTH_COOKIE_PATH=/tmp/dbauth.txt
  bash tests/workflows/genie/run-scenarios.sh --record

  # Step 4: Commit the artifacts
  git add tests/workflows/genie/recordings/INT-GN.*
  git commit -m "test: record Genie integration test INT-GN [helix-9ab3e99f]"
