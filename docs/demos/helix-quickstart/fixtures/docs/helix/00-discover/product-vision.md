# Product Vision: hello-helix

## Mission
Build a simple, reliable command-line temperature converter that developers can use for quick unit conversions in scripts, terminals, and automation workflows.

## Target User
Developers who need fast, scriptable temperature conversions without leaving the command line.

## Why It Matters
- **Speed**: Quick mental math breaks aren't needed; just run a command
- **Reliability**: Consistent, accurate conversions for scripting and documentation
- **Simplicity**: Single responsibility—do one thing well

## Scope
- Celsius ↔ Fahrenheit conversion only
- Plain text output suitable for piping and automation
- Local execution (no network dependency)

## Success Criteria
Users can quickly convert temperatures from the CLI without consulting external tools or calculators.
