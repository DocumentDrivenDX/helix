# Execution Docs

This directory reserves the canonical location for `EXEC-*` execution
documents in the HELIX artifact hierarchy.

Execution docs bridge test plans and executable tests or commands:

- `TP-*` defines what must be verified
- `EXEC-*` defines the bounded execution contract for running that verification
- executable tests or commands provide the runtime evidence

Until FEAT-011 / TD-011 implementation work lands, this directory may remain
sparsely populated.
