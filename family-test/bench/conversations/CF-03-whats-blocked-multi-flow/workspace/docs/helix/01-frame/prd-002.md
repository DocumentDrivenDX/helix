# PRD-002 — Order Reconciliation

## Problem

Operators reconcile yesterday's orders by hand each morning; the
process takes 45 minutes and misses ~3% of items.

## Solution

A nightly reconciliation pass that flags mismatches before the
morning standup.

## Open questions

- What's the cutoff time? (blocks acceptance criteria)
- Who owns flag triage? (blocks rollout plan)
