# Source Profile authoring prompt

Draft a Source Profile.
Sections in order: Source System, Schema Snapshot, Volume Estimates, Freshness Observed, PII Classification, Producers Contacted, Risks And Unknowns.
Use observed source facts, field names, volume ranges, contacts, and unresolved risks.
Put linkages to briefs, contracts, producer docs, or evidence in frontmatter `ddx.links`, not the body.
