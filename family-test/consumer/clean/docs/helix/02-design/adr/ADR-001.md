# ADR-001: Use PyYAML for slice validator

## Context
Stdlib lacks YAML; we want to prove design works fast.

## Decision Drivers
- Speed of iteration on the slice.
- Production validator will be stdlib-only per design.

## Considered Options
1. PyYAML for slice, hand-roll for production.
2. Hand-roll from day one.
3. JSON everywhere; drop YAML ergonomics.

## Decision
Option 1.

## Consequences
- Pro: slice ships in one session.
- Con: stdlib-only port is follow-up work.

## Status
Accepted (slice-scope).
